# so_long
2D game with X11
# How to play in Ubuntu?
Firstly, you should install X11. Follow these steps in terminal.
``` sudo apt-get update ```
``` sudo apt-get install -y x11-utils ```
``` make ```
``` ./so_long maps/map.ber ```
